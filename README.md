# Kelly Arquitetura

Landing page responsiva para apresentação do escritório, captação de leads e
exibição de projetos residenciais contemporâneos brasileiros.

## Recursos

- Hero com carrossel automático de quatro projetos
- Portfólio visual de residências modernas
- Seções de serviços, processo e apresentação profissional
- Formulário de captação de leads
- Chamada direta para WhatsApp
- Navegação responsiva e animações suaves

## Tecnologias

HTML, CSS e JavaScript, sem dependências de framework.

Para visualizar, abra `index.html` no navegador.
